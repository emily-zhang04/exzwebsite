A team-based database project from CNIT 37200: Database Programming, focused on designing a normalized relational schema for a flight reservation system, including an ERD to model entity relationships and constraints. Source-free Machine Unlearning With Low Rank Hessian Estimation On Tabular Data

Project from CNIT 57100: Responsible Data Science, extending recent source-free machine unlearning methods from vision to tabular data, addressing privacy-sensitive domains such as income, health, and credit records. Using logistic regression on the UCI Adult dataset, it introduces diagonal Hessian estimation via parameter perturbations and Newton-style updates to remove the influence of forgotten samples without access to original training data, while maintaining strong performance on retained data.
